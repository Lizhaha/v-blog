# v-blog

v-blog是一个利用vue.js编写的一个小型个人博客网站。所使用的主要技术及工具：
1. 脚手架工具vue-cli3
2. 利用axios完成与后台的交互
3. 利用vuex进行状态管理
4. 使用vue-router进行路由管理
5. 使用框架——iView

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
